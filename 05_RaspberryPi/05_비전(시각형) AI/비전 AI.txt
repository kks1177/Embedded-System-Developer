 - 분류 (Classification)
 - 객체 감지 (Object Detection)
 - 구분 (Segmentation)

분류 vs 객체 감지 차이점
 : 대상 + 위치

분류 vs 구분 차이점
 :  감지 + 위치 + 형태


데이터 수집 -> 데이터전처리 -> 데이터 학습 -> AI 모델 생성 -> AI 추론 검증 -> 데이터 수집 -> ... (반복 - 원하는 학습 모델 만들어질 때까지)


1. 분류 (Classification) 과정
(1) 데이터 준비 -> 데이터 수집 -> 데이터 레이블링(라벨링)
(2) 분류 학습하기 -> 학습 모델 생성
(3) 분류 추론하기 -> 학습 모델을 이용한 분류 및 예측

2. 객체 감지 학습 (Object Detection) 과정


3. 구분 (Segmentation) 과정
(1) 데이터 준비 -> 데이터 수집 -> 데이터 어노테이션
(2) 학습 데이터 제작하기 -> 어노테이션 파일 통합
(3) 분할 학습하기 -> 학습 모델 생성
(4) 분할 추론하기 -> 학습 모델을 이용한 객체 감지
